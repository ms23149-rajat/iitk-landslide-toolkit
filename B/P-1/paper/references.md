# Key Papers for TRIGRS v2.1

## Paper 1 — The Main Manual
**Citation:** Baum, R.L., Savage, W.Z., and Godt, J.W. (2008)
TRIGRS—A Fortran program for transient rainfall infiltration
and grid-based regional slope-stability analysis, version 2.0.
U.S. Geological Survey Open-File Report 2008-1159, 75 p.

**Download:**
https://pubs.usgs.gov/publication/ofr20081159

**What it contains:**
- Full theoretical basis of TRIGRS
- Complete user manual and initialization file guide
- Tutorial walkthrough (Appendix 2)
- All input/output file formats explained

## Paper 2 — The Parallel Version
**Citation:** Alvioli, M. and Baum, R.L. (2016)
Parallelization of the TRIGRS model for rainfall-induced
landslides using the message passing interface.
Environmental Modelling & Software 81, 122-135.

**Download:**
https://doi.org/10.1016/j.envsoft.2016.04.002

**What it contains:**
- Description of MPI parallelization (the prg program)
- Performance benchmarks on Colorado Front Range
- Details of TRIGRS v2.1 improvements over v2.0
